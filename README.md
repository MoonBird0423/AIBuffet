# AIBuffet

AI自助餐，新手用户学习AI的第一站

## 项目介绍
这是一个前后端分离的项目，前端使用React，后端使用Java Spring。

## 项目结构
- `/frontend` - React前端项目
- `/backend` - Java Spring后端项目

## 开发指南
1. 分别启动前端和后端服务
2. 前端开发命令：`cd frontend && npm start`
3. 后端开发命令：`cd backend && ./mvnw spring-boot:run`

## 部署说明
前后端可以分开部署：
- 前端可部署到静态Web服务器或CDN
- 后端可部署到Java应用服务器
