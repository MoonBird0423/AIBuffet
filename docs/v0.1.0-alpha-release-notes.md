# 书意平台 v0.1.0-alpha 预发布版本说明

## 🎉 版本概述

书意平台 v0.1.0-alpha 是项目的首个预发布版本，标志着"AI伴读，直达书魂"理念从概念到可用的重要里程碑。本版本专注于为新手用户提供AI学习的第一站体验，通过智能文档处理和知识管理，让用户能够快速构建个人知识库并进行智能对话。

**⚠️ 重要提示：这是一个预发布版本，可能存在一些已知问题和不稳定功能。建议在生产环境中谨慎使用。**

## ✨ 核心功能特性

### 🔐 用户认证系统
- **手机验证码登录**：简化注册流程，新用户自动创建账户
- **JWT身份认证**：安全可靠的用户会话管理
- **图形验证码防护**：防止恶意注册和登录攻击
- **用户信息管理**：支持头像上传、用户名修改等个性化设置

### 📚 知识库管理
- **个人知识库**：每个用户自动获得专属知识库
- **多格式文档支持**：支持PDF、DOCX、TXT、MD、EPUB、MOBI等格式
- **批量文档上传**：支持多文件同时上传，最大50MB单文件
- **文档处理状态**：实时显示文档处理进度和状态

### 🤖 AI智能处理
- **文本提取与分块**：自动从文档中提取文本并进行智能分块
- **向量化存储**：基于Milvus向量数据库的语义检索
- **多模型支持**：集成多种AI模型用于不同处理任务
- **异步处理队列**：支持大规模文档的并发处理

### 🔍 智能搜索系统
- **语义向量搜索**：基于文档内容的语义相似度检索
- **多维度搜索**：支持知识库搜索、文档搜索、多知识库搜索
- **权限控制**：精确的搜索权限管理，保护用户隐私
- **搜索结果优化**：智能排序和相关性评分

### 💬 智能对话系统
- **WebSocket实时通信**：支持流式对话响应
- **提问对象关联**：可针对特定图书或知识库进行对话
- **会话管理**：完整的聊天历史记录和会话状态管理
- **上下文感知**：基于文档内容的智能问答

### 📖 图书内容服务
- **公开图书馆**：已发布图书的公开浏览和搜索
- **图书详情页**：完整的图书信息展示
- **收藏功能**：用户可收藏感兴趣的图书到个人知识库
- **分类筛选**：按图书分类进行浏览和筛选

### 🎵 音频合成服务
- **AI语音解读**：为文档生成专业的语音解读
- **多角色音频**：支持不同角色的语音合成
- **音频状态管理**：实时监控音频生成进度
- **音频播放器**：内置音频播放和控制功能

### 🧠 高级内容生成
- **知识脑图**：自动生成文档的知识结构图
- **知识测试**：基于文档内容生成测试题目
- **内容解读**：AI生成的文档深度解读
- **导出功能**：支持脑图和测试题的导出保存

## 🛠 技术架构

### 后端技术栈
- **Spring Boot 3.x**：现代化的Java应用框架
- **Spring Security**：完善的安全认证体系
- **Spring Data JPA**：数据持久化层
- **Milvus向量数据库**：高性能向量检索
- **MySQL**：关系型数据存储
- **Redis**：缓存和会话管理
- **阿里云OSS**：对象存储服务

### 前端技术栈
- **React 18**：现代化的前端框架
- **Tailwind CSS**：原子化CSS框架
- **WebSocket**：实时通信支持
- **React Router**：单页应用路由
- **Context API**：状态管理

### 部署架构
- **前后端分离**：独立部署和扩展
- **Nginx反向代理**：负载均衡和静态资源服务
- **Docker容器化**：便于部署和环境管理
- **环境配置**：支持开发、测试、生产环境

## 🔒 安全特性

- **JWT令牌认证**：无状态的用户认证机制
- **权限分级控制**：公开内容和私有内容的精确权限管理
- **输入验证**：完善的参数验证和SQL注入防护
- **文件上传安全**：文件类型验证和大小限制
- **API访问控制**：基于角色的API访问权限

## 📊 性能优化

- **数据库索引优化**：关键查询的性能优化
- **缓存策略**：Redis缓存减少数据库压力
- **异步处理**：文档处理的异步队列机制
- **分页查询**：大数据集的分页加载
- **CDN加速**：静态资源的CDN分发

## 🚀 部署说明

### 环境要求
- Java 17+
- Node.js 16+
- MySQL 8.0+
- Redis 6.0+
- Milvus 2.3+

### 快速启动
```bash
# 后端启动
cd backend && ./mvnw spring-boot:run

# 前端启动
cd frontend && npm start
```

## ⚠️ 已知问题

### v0.1.0-alpha版本限制
- **音频生成功能**：需要阿里云语音服务配置，当前可能不稳定
- **大文件上传**：可能需要较长处理时间，建议文件大小控制在50MB以内
- **WebSocket连接**：在某些网络环境下可能不稳定，建议使用稳定的网络环境
- **向量搜索性能**：在数据量大时可能需要优化，当前版本建议知识库文档数量控制在1000个以内

### 性能考虑
- **首次加载**：可能需要较长时间，建议耐心等待
- **大量文档处理**：同时处理多个文档时可能影响响应速度
- **内存使用**：处理大文档时内存使用较高

### 兼容性说明
- **浏览器支持**：建议使用Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- **移动端**：当前版本主要针对桌面端优化，移动端体验可能不够完善
- **网络要求**：建议使用稳定的网络连接，避免在网络不稳定时进行重要操作
