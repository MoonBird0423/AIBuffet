# 首页改进计划

## 目标
移除多选比较功能，优化页面布局，提升用户体验。

## 具体修改

### 1. ModelCard 组件修改
```diff
- 移除 props：isSelected, onSelect
- 删除复选框 UI
- 删除"加入对比"按钮
- 删除选中状态相关的样式
+ 优化卡片内部间距和布局
```

### 2. Home 组件修改
```diff
- 移除状态：selectedModels
- 移除函数：handleModelSelect, handleComparisonClick
- 删除提示文字："选择最多3个模型进行比较"
- 删除比较按钮及其容器
- 删除已选择模型统计
+ 增加卡片网格间距
+ 优化标题和内容间距
```

### 3. 布局优化
1. 调整卡片网格间距：
   - 将 gap-8 改为 gap-10，增加卡片之间的间距
   - 调整卡片内边距，确保内容更加紧凑

2. 优化标题区域：
   - 增加标题下方的间距
   - 移除多余的提示文字

## 预期效果
- 页面更加简洁清晰
- 重点突出模型展示
- 用户可以更快地找到并使用感兴趣的模型

1. 修改 ModelCard.js
2. 修改 Home.js
3. 测试页面布局和响应式设计
4. 确认所有功能正常工作
